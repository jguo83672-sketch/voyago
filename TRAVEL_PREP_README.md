# 旅行准备模块功能说明

## 概述

本次更新为 Voyago 平台增加了完整的旅行准备模块，包含两个核心子系统：

1. **出行预订系统** - 酒店和机票推荐与预订
2. **旅行用品商店** - 联名旅行用品电商平台

---

## 功能清单

### 一、出行预订系统

#### 1.1 数据模型
- **TravelPrep** - 旅行准备记录
  - 关联行程和用户
  - 保存入住/退房日期、人数、预算等信息

- **HotelRecommendation** - 酒店推荐记录
  - 保存推荐酒店详情（名称、评分、价格、设施等）
  - 包含预订链接

- **FlightRecommendation** - 机票推荐记录
  - 保存推荐机票详情（航班号、时间、价格等）
  - 包含预订链接

#### 1.2 核心功能
- **行程准备页面** (`/itinerary/<id>/prepare`)
  - 填写准备信息（入住/退房日期、人数、预算）
  - 查看准备清单（证件类、衣物类、洗漱用品、电子设备、药品类等）
  - 获取酒店推荐
  - 获取机票推荐

- **酒店推荐功能**
  - 根据目的地城市、入住日期、预算智能推荐酒店
  - 显示酒店评分、价格、设施、位置信息
  - 提供预订链接（跳转至携程等平台）

- **机票推荐功能**
  - 根据出发地、目的地、出行日期推荐航班
  - 显示航班号、时间、价格、舱位信息
  - 提供预订链接

#### 1.3 智能推荐服务 (`travel_prep.py`)
- 内置模拟酒店数据库（北京、上海、广州、东京等城市）
- 内置模拟机票数据库（北京-上海、北京-东京、广州-东京等航线）
- 智能城市匹配算法
- 可扩展接入真实API（携程、飞猪等）

#### 1.4 准备清单
- 自动根据行程天数生成准备清单
- 包含：证件类、衣物类、洗漱用品、电子设备、药品类、其他
- 区分必带和可选物品
- 支持勾选记录

---

### 二、旅行用品商店

#### 2.1 数据模型
- **TravelProduct** - 商品
  - 商品基本信息（名称、描述、分类）
  - 价格信息（现价、原价、折扣）
  - 库存和销量
  - 评分和评论数

- **ProductReview** - 商品评论
  - 用户评分和评论内容
  - 支持图片上传

- **CartItem** - 购物车
  - 记录用户购物车商品

- **Order** - 订单
  - 订单号、收货信息
  - 支付方式和状态
  - 物流信息

- **OrderItem** - 订单商品
  - 订单商品快照

#### 2.2 核心功能

##### 2.2.1 商城首页 (`/travel-shop`)
- 精选商品展示
- 分类浏览（行李箱、洗漱包、一次性用品、转换插头、电话卡等）
- 多种排序方式（价格、销量、评分）
- 分页加载

##### 2.2.2 商品详情 (`/travel-shop/product/<id>`)
- 商品详细信息和图片
- 价格和折扣显示
- 评分和评论展示
- 相关商品推荐
- 添加到购物车/收藏

##### 2.2.3 购物车 (`/cart`)
- 查看购物车商品
- 调整商品数量
- 删除商品
- 自动计算运费（满200免运费）
- 去结算

##### 2.2.4 结账 (`/checkout`)
- 填写收货地址
- 选择支付方式（微信/支付宝/银行卡）
- 确认订单信息
- 提交订单

##### 2.2.5 订单管理
- 订单列表 (`/orders`)
  - 按状态筛选（待付款、待发货、待收货、已完成）
  - 分页显示
- 订单详情 (`/order/<id>`)
  - 订单完整信息
  - 支付状态
  - 物流信息
  - 评价订单

#### 2.3 商品分类
- **行李箱**：登机箱、中号行李箱、大号行李箱
- **洗漱包**：洗漱包、化妆包
- **一次性用品**：洗漱套装、拖鞋、毛巾
- **转换插头**：万能转换插头、欧洲插头、美式插头
- **电话卡**：日本卡、韩国卡、欧洲卡
- **其他**：旅行枕、套装等

## 使用说明

### 初始化商品数据
```bash
python init_products.py
```

### 访问路径

#### 出行预订系统
1. 查看行程详情
2. 点击"开始准备"按钮
3. 填写准备信息
4. 查看酒店/机票推荐
5. 点击预订链接跳转

#### 旅行用品商店
1. 导航栏 → "旅行商店"
2. 浏览商品分类
3. 查看商品详情
4. 加入购物车
5. 去购物车结算
6. 填写地址并支付
7. 在"我的订单"中查看订单状态

## 注意事项

1. **权限控制**：只有行程作者可以查看准备页面
2. **价格显示**：所有价格为人民币（CNY）
3. **库存管理**：下单时自动扣减库存
4. **购物车数量**：导航栏显示购物车数量徽章
5. **订单号**：自动生成唯一订单号
