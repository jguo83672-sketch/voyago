# 更新日志

## [1.0.0] - 2026年3月

### 新增功能

#### 目的地系统
- ✨ 新增邮轮游分类（region_type: cruise）
- ✨ 新增周末近郊游分类（region_type: weekend）
- ✨ 新增主题乐园分类（region_type: theme）
- 🎨 优化目的地分类卡片，支持一行内显示 5 个分类
- 🎨 为邮轮游、周末近郊游、主题乐园设计独特的图标和颜色

#### OCR 智能导入
- ✨ 集成百度智能云 OCR 文字识别
- ✨ 集成 DeepSeek AI 智能分类
- ✨ 支持 16 种攻略类型自动分类
- ✨ 自动生成攻略标题
- ✨ 自动提取目的地信息
- 📝 添加 OCR 导入使用说明文档

#### 旅行准备系统
- ✨ 新增出行预订功能
  - 酒店推荐与预订（模拟数据）
  - 机票推荐与预订（模拟数据）
- ✨ 新增旅行准备清单
  - 自动生成物品清单（证件、衣物、洗漱、电子、药品）
  - 支持勾选记录
- ✨ 为每个行程提供专属准备页面

#### 旅行用品电商
- ✨ 新增完整电商系统
  - 商品展示和分类浏览
  - 商品详情、评分和评论
  - 购物车管理
  - 订单系统（支付、发货、完成）
  - 运费计算（满 200 免运费）
- 📦 初始化商品数据（行李箱、洗漱包、转换插头等）

#### 社群系统
- ✨ 新增旅行社群功能
  - 创建和管理社群
  - 成员管理和权限控制
  - 发起和管理线下活动
  - 活动报名和参与者管理

#### 旅行足迹
- ✨ 新增旅行足迹记录
  - 记录去过的目的地和访问日期
  - 添加旅行笔记和照片
  - 目的地评分

### 优化改进

#### 前端优化
- 🎨 目的地分类卡片改为垂直布局，更适合一行显示
- 🎨 优化图标大小和间距，提升视觉一致性
- 📱 增强响应式设计，适配多种屏幕尺寸

- 🔧 添加搜索服务（web_search.py）

#### 文档完善

### 技术栈

| 技术 | 用途 |
|------|------|
| Flask 3.0.0 | Web 框架 |
| SQLAlchemy | ORM |
| SQLite / PostgreSQL | 数据库 |
| Bootstrap 5 | 前端 UI |
| 百度智能云 OCR | 文字识别 |
| DeepSeek API | AI 分类和规划 |
| Google Custom Search API | 搜索服务 |

### 已知问题

- 当前酒店和机票推荐使用模拟数据，需要接入真实 API
- OCR 识别对于复杂排版或手写文字可能不够准确
- 部分功能需要用户手动确认 AI 生成的内容
