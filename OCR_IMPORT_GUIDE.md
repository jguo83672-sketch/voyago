# 攻略 OCR 导入功能使用说明

## 功能概述

Voyago 支持通过百度 OCR 技术自动识别上传的攻略图片中的文字内容，并使用 DeepSeek AI 进行智能分类和整理，帮助用户快速导入来自其他平台的攻略内容。

## 功能特点

1. **文字识别**：使用百度智能云 OCR 技术，高精度识别图片中的文字
2. **自动分类**：AI 自动识别攻略类型（美食、交通、住宿、景点等）
3. **智能标题**：根据内容自动生成吸引人的标题
4. **目的地识别**：自动提取攻略中的目的地信息
5. **图片预览**：上传后即时预览图片
6. **手动编辑**：识别后可手动修改和补充内容

## 使用步骤

### 1. 访问 OCR 导入页面

方式一：通过导航栏
- 登录后，点击右上角用户名
- 选择"OCR 导入攻略"

方式二：通过发布攻略页面
- 进入"发布攻略"页面
- 点击右上角的"OCR 导入"按钮

### 2. 上传攻略图片

- 点击"选择攻略图片"按钮
- 选择来自其他平台的攻略截图（携程、马蜂窝、小红书等）
- 支持 JPG、PNG、JPEG 等常见格式

### 3. 自动识别和分类

- 系统自动调用百度 OCR 识别图片中的文字
- DeepSeek AI 自动分类攻略类型
- 自动生成建议标题
- 提取目的地信息

### 4. 编辑和发布

- 检查识别的文字内容
- 修改标题、分类、目的地
- 补充或修改攻略内容
- 选择封面图片
- 点击"发布攻略"

## 使用技巧

### 提高识别准确率

1. **图片清晰度**：确保图片清晰，文字清晰可见
2. **避免反光**：避免图片上有强烈反光
3. **光线充足**：选择光线良好的图片
4. **文字大小**：文字不要太小，建议使用高清截图
5. **图片格式**：优先使用 JPG 或 PNG 格式

### 最佳实践

1. **截图技巧**：使用系统截图工具，选择"窗口"或"区域"截图
2. **多图片处理**：对于长攻略，建议分多次上传，或使用高精度截图
3. **内容补充**：OCR 识别后，建议人工检查并补充细节
4. **分类调整**：系统自动分类可能不够准确，建议手动确认
5. **标题优化**：可以修改自动生成的标题，使其更具吸引力

## 技术架构

### 后端组件

- **OCR 服务** (`ocr_service.py`)：百度智能云文字识别
- **分类服务** (`guide_classifier.py`)：DeepSeek AI 智能分类
- **路由处理** (`routes.py`)：OCR 导入和分析接口

### 前端组件

- **导入页面** (`import_guide_ocr.html`)：图片上传和识别界面
- **创建攻略** (`create_guide.html`)：内容编辑和发布界面

### API 调用流程

1. 用户上传图片 → OCR 识别文字
2. 识别结果 → DeepSeek AI 分类和标题生成
3. 返回结果 → 填充到表单
4. 用户编辑 → 发布攻略

## 配置说明

### 百度 OCR 配置

在 `.env` 文件中配置：

```env
BAIDU_OCR_API_KEY=your-baidu-ocr-api-key-here
BAIDU_OCR_SECRET_KEY=your-baidu-ocr-secret-key-here
```

获取方式：
1. 访问 [百度智能云](https://cloud.baidu.com/)
2. 创建应用并开通文字识别服务
3. 在控制台获取 API Key 和 Secret Key

### DeepSeek AI 配置

在 `.env` 文件中配置：

```env
DEEPSEEK_API_KEY=your-deepseek-api-key-here
```

获取方式：
1. 访问 [DeepSeek 平台](https://platform.deepseek.com/)
2. 注册并获取 API Key

## 注意事项

1. **图片大小限制**：单张图片最大 32MB
2. **识别时间**：根据图片大小和内容复杂度，识别可能需要 3-10 秒
3. **API 限制**：百度 OCR 有 QPS 限制，请注意不要频繁调用
4. **内容检查**：OCR 识别可能有误差，务必人工检查后再发布
5. **版权注意**：导入的攻略内容请注意版权问题，建议添加原创说明

## 常见问题

### Q: 识别出的文字有乱码或错误怎么办？
A: OCR 识别可能产生误差，建议在编辑页面手动修正后再发布。

### Q: 自动分类不准确怎么办？
A: 可以手动修改分类选择更合适的类型。

### Q: 可以一次上传多张图片吗？
A: 目前支持单张图片上传，多张图片需要分别识别后合并。

### Q: 识别失败怎么办？
A: 请检查：
- 图片是否清晰
- 网络连接是否正常
- API 配置是否正确

### Q: 生成的标题不合适怎么办？
A: 可以在编辑页面手动修改标题，不受限制。
